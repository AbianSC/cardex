# CARDEX — Ionic + Express + Sequelize + MySQL

“Log it. Find it. Drive it.”

## ¿Qué es?
App móvil para gestionar coches de ocasión con 3 CRUD:
- **Cars** (coches)
- **Customers** (clientes)
- **Inquiries** (consultas de un cliente sobre un coche)


## PUESTA EN MARCHA:

### Backend y Frontend
bash
cd backend
node index.js
# Servidor: http://localhost:8080


cd frontend
ionic serve
# App: http://localhost:8100


### RUTAS

Home: http://localhost:8100/home

Cars: http://localhost:8100/my-cars

Customers: http://localhost:8100/customers

Inquiries: http://localhost:8100/inquiries
